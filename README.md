Kisan Sahayata 🌾

Kisan Sahayata is a modern, AI-powered web application designed to help farmers track, verify, and understand their subsidy eligibility. The platform ensures transparency, accessibility, and ease of use, enabling farmers to stay informed and empowered.

🚀 Features
Subsidy Status Tracking – Farmers can check the status of government subsidies in real-time. </br>
Verification & Authenticity – QR codes are generated for secure verification of subsidy records.</br>
AI-Assisted Guidance – Google Gemini (2.5 Flash) provides eligibility guidance and drafts multilingual reports.</br>
Email & Notification Support – Alerts users about subsidy updates and verification results.</br>
Data Visualization – Interactive dashboards show subsidy history, status, and trends.</br>
User-Friendly & Responsive UI – Designed with modern, accessible components for all devices.</br>
🛠️ Technology Stack</br>
Frontend: Next.js 15 (App Router) for server-side rendering and routing, TypeScript for type safety, Tailwind CSS for responsive styling.</br>
UI Components: ShadCN UI (Radix UI primitives) and Lucide React for icons.</br>
State & Forms: React Hook Form for complex form management, Zod for schema validation.</br>
Backend / Database: Firebase Firestore for real-time data, Firebase Authentication for secure identity management (anonymous/ID-based sessions).</br>
AI Integration: Genkit framework with Google Gemini (2.5 Flash) for eligibility guidance and report generation.</br>
Visualization & QR Codes: Recharts for dashboards and QRCode.react for generating secure authenticity codes.</br>
📦 Installation & Setup

Clone the repository:

git clone <your-repo-link>

Install dependencies:

npm install

Add Firebase configuration in the project:

const app = initializeApp(firebaseConfig);

Run the project locally:

npm run dev
